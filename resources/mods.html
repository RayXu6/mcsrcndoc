合法模组 #

警告

MC速通只允许加入这里的合法模组
否则将不过审
速通只允许使用Fabric Loader(对于1.14以前的版本 只允许使用Legacy Fabric)
不允许使用Fabric API
不允许使用Forge Loader 请尽量使用Java17及以上

Sodium #

1.15.2 | 1.16.1 | 1.16.5 | 1.17.1 | 1.18.2 | 1.19
重写渲染引擎 优化游戏体验

注意

冲突的模组: Optifine

Optifine #

1.14-
老牌优化模组

注意

依赖Optifabric   
对于Optifine有单独的[规则](http://bombch.us/DOOK)(谷歌文档)或加群查看如何设置

Optifabric #

1.13- | 1.14.4
使Optifine可以运行在Fabric上

Lithium #

1.15.2 | 1.16.1 | 1.16.5 | 1.17.1 | 1.18.2 | 1.19.2
优化了内置服务端

Starlight #

1.16.1/1.16.5/1.17.1 | 1.18.2 | 1.19-1.19.3
重写光照机制以达到修复光照错误以及提高帧数的作用 ::: warning 注意 冲突的模组: Phosphor (更推荐Starlight)
1.15.2版本的请加群下载 :::

Phosphor #

1.15.2-1.19.3
优化光照机制 ::: warning 注意 冲突的模组: Starlight (更推荐Starlight)
:::

Krypton #

1.15.2-1.19.3
优化网络和实体问题 ::: warning 注意 不推荐装入 可能导致部分宏出现bug
:::

Dynamic FPS #

1.7-1.8.9 | 1.15.2 | 1.16.1
当MC在后台时锁帧 减少性能消耗 ::: warning 注意 冲突的模组: SleepBackground (更推荐SleepBackground)
不建议单开使用 多开建议使用 :::

ServerSideRNG #

1.16.1
反作弊模组

SpeedRunIGT #

1.7.2-1.19.3
计时器模组

Atum #

1.7.10 | 1.8 | 1.8.9 | 1.9.4-1.13.2 | 1.14.4-1.19.3
RSG/SSG/FSG的快速重开模组

注意

对于1.9.4-1.13.2的版本 请使用1.0.5的Atum 否则会有bug

Fast Reset/Anchiale #

1.8.9 | 1.14.4 | 1.15.2-1.17.1
跳过保存世界直接重开

注意

如果你不小心手滑在游玩中按到了本模组的重开按钮 那么将会部分回档

WorldPreview #

1.8.9 | 1.14.4-1.17.1 | 1.18.1-1.19.3
世界预览模组

注意

1.18.1-1.19.3版本的WorldPreview需自行构建
- 对于1.14.4的WorldPreview 如果想与Optifine共同使用 请使用2.2.5的版本

LazyDFU #

1.14.4-1.18.2 | 1.19.x
加快加载速度

Voyager #

1.14.x-1.16.x
修复了一个罕见的bug(ConcurrentModificationException)

Force Port #

1.13- | 1.14+
将对局域网开放的端口设为25565 供多人速通使用

FabricProxy-Lite #

1.16+
可以接入Velocity服务端(对于速通没啥用)

LazyStronghold #

1.14.4-1.17.1
将要塞生成放在一个新线程上 加快加载速度

BiomeThreadLocalFix #

1.16.5+
修复了群系引起的内存溢出

SleepBackground #

1.7+
当MC在后台时锁帧 减少性能消耗 ::: warning 注意 冲突的模组: Dynamic FPS (更推荐SleepBackground)
不建议单开使用 多开建议使用 :::

antiresourcereload #

1.13.2 | 1.14.4 | 1.15.2 | 1.16.1 | 1.16.5 | 1.17.1-1.18.1
让世界加载更快

StandardSettings #

1.7+
可以进行默认设置 每次重开是自动调为默认设置

Tab Focus #

1.13.2-

LegacyPlanarFog #

1.14.4-
老版本的2D迷雾

StatsPerWorld #

1.3.2-1.6.4
将成就独立到每个世界

Extra Options #

1.14.4-1.16.1
在旧版本加入了1.16.2+的关闭FOV效果以及黑白徽标

警告

仅提供医学证明确有需要者 才可使用 否则speedrun.com无法过审

Set Spawn #

1.6.4 | 1.7.10 | 1.8/1.8.9 | 1.15.2 | 1.16.1 | 1.16.5 | 1.17.1
将你的出生点固定到你设置的坐标

注意

仅限SSG/SS使用 大部分项目的主流种子和setspawn配置将被写在SSG种子里

ChunkCacher #

1.14-1.15.2 | 1.16-1.17.1 | 1.18-1.18.2
从第二个世界开始加快加载速度

注意

仅限SSG/SS使用
依赖Atum
1.14-1.15.2的版本仅与WorldPreview冲突
1.16-1.17.1以及1.18-1.18.2的版本与WorldPreview以及LazyStronghold冲突

小项目的模组 #

  • Pogloot
    使你拥有极致运气
  • Pogworld
    改变世界生成 满满的全是结构
  • Icarus
    开局自带鞘翅及64个烟花火箭
  • HHH
    开局只有半颗心 并且为极限模式
  • Infiniperal
    开局自带一个无伤的无限珍珠
  • Infinity Icarus
    Icarus 但是烟花火箭是无限的
  • Netherless
    开局有12颗末影之眼 无法进入下界
  • Tyche
    相当于数据包 可以控制SSG的RNG
